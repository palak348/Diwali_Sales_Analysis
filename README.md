# Diwali Sales Analysis

A comprehensive data analysis project examining Diwali sales patterns and customer behavior.

## Overview
This project analyzes Diwali sales data to uncover insights about customer demographics, purchasing patterns, and product preferences.

## Key Findings
- **Gender**: Female customers show higher purchasing power
- **Age Group**: 26-35 years age group dominates sales
- **Top States**: Uttar Pradesh, Maharashtra, and Karnataka lead in orders
- **Popular Categories**: Food, Clothing, and Electronics are top-selling products

## Files
- `Diwali_Sales_Analysis.ipynb` - Main analysis notebook
- `Diwali Sales Data.csv` - Dataset containing sales information

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Includes
- Data cleaning and preprocessing
- Gender-based purchasing analysis
- Age group distribution
- State-wise sales comparison
- Marital status impact
- Occupation patterns
- Product category performance
